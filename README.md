# Install 
- Clone Repo
```
git clone https://github.com/KelompokMVCAdamPlorenNathanYesa/utpas-social/
```
# Install Depedencies
```
npm i
```
- Jalanin Node Server (Buat TailWind css) untuk  develop saja kalau tidak lanjut step selanjutnya
```
npm run dev
```
- Jalanin php server
```
php -S 127.0.0.1:8000
```
